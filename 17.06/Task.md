# Практическое задание: Базовая стилизация и блочная модель

## Цель задания

Подключить внешнюю таблицу стилей к HTML-каркасу, освоить работу с классами, подобрать уникальную типографику и палитру, а также применить правила блочной модели (`margin`, `padding`, `border`) для создания индивидуального дизайна страницы.

## Техническое задание

### Шаг 1. Подключение стилей и выбор шрифта

1. В директории `MyPortfolio` создайте файл `style.css`.
2. В файле `index.html` внутри тега `<head>` подключите созданный CSS-файл с помощью тега `<link>`.
3. Зайдите на сервис [Google Fonts](https://fonts.google.com/), выберите любой понравившийся вам шрифт и подключите его к сайту.
4. В файле `style.css` обратитесь к селектору `body`, примените выбранный шрифт и обязательно **сбросьте стандартные внешние отступы**.

### Шаг 2. Разметка классами в HTML

Откройте ваш `index.html` и выдайте логическим блокам уникальные классы, названия для которых **придумайте сами** (используйте английский язык, например: `.my-header`, `.about-section`, `.hobby-card`).

* Обязательно дайте классы шапке, футеру и всем смысловым секциям внутри `<main>`.

### Шаг 3. Цветовая палитра и Типографика

Стилизуйте элементы, обращаясь к ним в CSS *только через селекторы классов* (с точкой):

1. Подберите для своего сайта цветовую палитру (2-3 цвета). Можно использовать сервис [Coolors](https://coolors.co/).
2. Задайте цвет фона (`background-color`) и цвет текста (`color`) для вашей шапки и футера. Они должны контрастировать с основной частью сайта.
3. Измените стили для главного заголовка `<h1>`: задайте ему выравнивание по центру, уникальный цвет и подберите комфортный размер шрифта (например, от `28px` до `40px`).
4. Измените цвет всех ссылок навигации на любой, кроме стандартного синего. Уберите у них стандартное нижнее подчеркивание.

### Шаг 4. Блочная модель (Отступы и рамки)

Используйте блочную модель, чтобы добавить интерфейсу "воздуха":

1. Задайте внутренние отступы (`padding`) для вашей шапки и подвала.
2. Добавьте `padding` для всех секций внутри `<main>`, чтобы текст внутри них смотрелся гармонично.
3. Чтобы секции не слипались друг с другом по вертикали, задайте им внешний отступ снизу (`margin-bottom`). Размер отступа выберите на свой вкус.
4. **Стилизация фото:** Оберните вашу фотографию (`img`) в рамку с помощью свойства `border`. Выберите цвет и толщину рамки.
5. Задайте фотографии фиксированную ширину (`width`) от `150px` до `300px` на ваш выбор.

6. *Дополнительно:* Попробуйте скруглить углы у вашей фотографии с помощью свойства `border-radius` (например, `15px` или `50%` для круга).

## Требования к выполнению

1. В CSS-файле не должно быть стилизации классов, которых нет в HTML-файле.
2. Не использовать встроенные стили (атрибут `style` внутри HTML-тегов категорически запрещен).
3. Все отступы и размеры должны быть заданы с указанием единиц измерения (`px`).
4. После обновления страницы в браузере элементы не должны "слипаться" друг с другом.
